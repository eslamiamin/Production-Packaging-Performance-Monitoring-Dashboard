# Production-Packaging-Performance-Monitoring-Dashboard
Due to data confidentiality policies, all datasets used in the portfolio version of this project have been anonymized and modified. The data structure, logic, KPIs, and modeling approach remain consistent with the original implementation, while sensitive information has been removed or altered.

Led the design and implementation of a performance monitoring dashboard for Production & Packaging operations


Established a structured data foundation by collaborating with ERP and system leadership teams


Redesigned data entry workflows to improve data accuracy and governance


Performed continuous reconciliation between production, warehouse, and sales datasets


Modeled and implemented KPIs including productivity metrics, defect rates, plan achievement, and labor efficiency indicators


Conducted multi-month validation and refinement of performance metrics


Delivered executive-ready dashboards used in senior management meetings


Enabled data-driven performance evaluation and incentive system implementation
